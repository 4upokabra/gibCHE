# Roadmap ReconScope

## Release 0.3 · декабрь 2025
- Расширяем разведку: subdomain-enum, пассивный DNS-хантер, интеграции с leak-источниками.
- Новые шаблоны атак (RDP/SMTP brute, десериализация, обёртки для популярных PoC).
- Улучшение LLM-пайплайна: дополнительные доменные подсказки, снижение «галлюцинаций», ускоренный Playwright-парсинг.

## Release 0.4 · февраль 2026
- AutoPentest: поддержка цепочек pivoting/lateral movement и визуализация attack-path.
- LLM Learning Loop: дообучение на собственных отчётах, автоматический feedback.
- **Chat Pentest (beta)** — чат с LLM для общения, запуска разведки/эксплойтов и получения результатов.

## Release 0.5 · май 2026
- Полноценный режим Chat Pentest: сохранение сессий, коллаборация нескольких исследователей, голосовые подсказки.
- Новый пакет сканеров: облачные конфигурации (AWS/GCP/Azure), IoT/ICS профили, ML-анализ аномалий.
- Attack Engine Autopilot: предложения LLM автоматически ставятся в очередь с валидацией и журналированием.

## Release 0.6 · август 2026
- LLM Plugins: внешние модели/промпты и приватные RAG-базы компаний.
- Расширение атак: контейнерные эксплойты, OT/SCADA сценарии.
- Chat Pentest + Ops: создание тикетов, запуск длительного мониторинга и обмен артефактами прямо из чата.
